Увеличьте свой словарный запас с RSLang.

О приложении.

RSLang – позволяет пользователю самому выбирать слова для изучения из коллекции 3600 слов, разбитой на шесть групп. Процесс изучения полностью регулируется пользователем в разделе "Учебник". Есть возможность удалять и помечать слова, как сложные, а также можно закрепить изучаемые слова в мини-играх.
В разделе "Словарь" пользователь может посмотреть слова выбранные на изучение и задействованные в мини-играх по разделам "Изучаемые слова", "Сложные слова" и "Удалённые слова".
Благодаря разделу "Статистика" можно отслеживать свой прогресс изучения.
